Welcome to Prowl Alfred Workflow

This Alfred 2 workflow allow you to send Prowl alerts to your mobile phone.

It requires Alfred 2 and the Powerpack plus Prowl.

Installation

To install, simply download the workflow (the prowl.alfredworkflow file) and then double-click it in the finder. It will open Alfred and ask you to allow it to be installed.

Configuration

To configure this workflow for the first time (or anytime you change you API key), type "prowl" followed by your API key (found under the API Keys tab and then pressing enter WHILE HOLDING THE COMMAND KEY. When pressing the command key, you should see the action description change to "Set Prowl API Key."

When setting a new key, the workflow will verify it first. Should there be a problem with verifying the key, a notification saying "_No APIKEY. Please configure by holding down the cmd key and pasting in prowl APIKEY._" will be shown.

Usage

To use, simply type the keyword "prowl" and your message to be sent to your mobile (separated by a space). Your alert will be sent to your mobile phone at prowl priority "Normal."

A notification will be shown on the screen showing if the alert was sent successfully or not.
